# Wildfire Risk Modeling Exercise Data
The data required for the exercise must be downloaded from the following links into the data folder. Alternatively, all the data can be downloaded at once from [here](https://1drv.ms/u/c/2c8b7dcd0ff009a3/IQB8o3ZuzEXCQJhdYC9NAiD_AaY7MKzMsZWkSo5e-4ceMiE?e=G0allt).

## [Wildfire Commons Data](https://1drv.ms/u/c/2c8b7dcd0ff009a3/IQD7y7TcRffvRZ_12YYmPNN-AbkzOJl3k8X5PP7nT_bBW5A?e=6jyCG9)
This folder contains the base Wildfire Commons input data used to initialize and run the WRF-SFIRE simulations. It includes the core atmospheric, fire, fuel, and domain-related datasets required by the exercise. These files provide the baseline configuration from which the ensemble simulations and perturbation experiments are derived.

## [Extra Data](https://1drv.ms/u/c/2c8b7dcd0ff009a3/IQCyyauHjxAAS5XuTOYCHM8dAayg_kXH8_qmrCDtFEIjKSc?e=sihwQc)
This folder contains supplemental datasets used to support the workflow but not directly produced by the main processing scripts. These data may include additional geospatial inputs, reference layers, configuration files, or auxiliary products needed for visualization, comparison, or manual inspection of the Lake Tahoe wildfire risk case.

## [Processed Data](https://1drv.ms/u/c/2c8b7dcd0ff009a3/IQDKujlzytivSIzqW46NFev7AeDe6s6BYiJY7pyNNBRhePE?e=gjzsi8)
This folder contains intermediate data generated after preprocessing the WRF-SFIRE inputs and outputs. It includes cleaned, compressed, or reformatted datasets used to reduce storage requirements and simplify analysis. Examples include processed fire progression fields, ensemble probability products, and geospatial data prepared for plotting or export.

## [Output Data](https://1drv.ms/u/c/2c8b7dcd0ff009a3/IQC61qY2e9A_RIUR2A4s6hV8AT-D5xyUcMTD_X3Il16wY0o?e=NjUyGO)
This folder contains the final products generated by the wildfire risk modeling workflow. These outputs include the ensemble fire probability layers, hourly fire progression products, classified GeoPackage files, and visualization-ready datasets. These files are intended for final analysis, mapping, and optional upload to ArcGIS Online.
